# Gmsh Language for VS-Code

## Install

Clone this repo into `~/.vscode/extensions/` and relaunch VSCode:

```
cd ~/.vscode/extensions/
git clone git@github.com:Bertbk/vscode-gmsh.git
```

## Features

- Color syntaxing for GMSH `.geo` file
- Automate indent
